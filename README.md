# Academic Events Portal

Starter project scaffold for SEN4001 WRIT1.

## Included modules (mapped to brief)
- Site map
- Home page
- Registration page
- Login page
- Reset password page
- Events listing page
- Events adding page
- MySQL/MariaDB schema
- Backend API structure (Node.js + Express)
- Report template

## Project layout
- `frontend/`: HTML, CSS, JavaScript UI
- `backend/`: API server and database integration
- `database/`: SQL schema and seed scripts
- `docs/`: site map and report template
- `tests/`: API and manual test notes

## Quick start (after implementation)
1. Configure `.env` from `backend/.env.example`
2. Create database using `database/schema.sql`
3. Start backend server
4. Open frontend pages in browser or serve statically via backend
